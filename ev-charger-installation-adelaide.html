Frequently Asked Questions

How long does EV charger installation take?

Most installations completed in 4-6 hours. Simple external runs can be faster, complex concealed runs may take longer. All work completed same day.

Do I need 3-phase power for an EV charger?

No, but it's faster. Single phase = 7kW (adds ~35km range per hour). 3-phase = 22kW (adds ~110km range per hour). Most homes have single phase, which is fine for overnight charging.

Will it work with my electric vehicle?

Yes. Tesla Wall Connector works with ALL EVs (comes with Type 2 cable). Universal Type 2 chargers also work with all EVs. Both use the Australian Type 2 standard.

How much does it cost to run an EV charger?

Electricity only. Example: Tesla Model 3 (50kWh battery) costs ~$12-15 to fully charge (off-peak rates). Compare to $80-100 petrol fillup. Huge savings.

Do you install at apartments/units?

Yes, if you have dedicated parking and can access the switchboard. Body corporate approval usually required. I can provide technical specs for your BC application.

What if my switchboard is full?

I'll assess capacity and recommend upgrades if needed. Sometimes requires larger switchboard or moving circuits. I'll quote all options.
